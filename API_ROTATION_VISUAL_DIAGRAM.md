### Retry Time Calculation

```
For contact with R retries:

Base time = API_LATENCY + DELAY
Retry time = R Ã— (API_LATENCY + RETRY_WAIT)

With values:
  API_LATENCY = 2s
  DELAY = 0.5s
  RETRY_WAIT = 2s

Success on 1st attempt: 2 + 0.5 = 2.5s
Success on 2nd attempt: 2 + 2 + 2 + 0.5 = 6.5s (+4s)
Success on 3rd attempt: 2 + 2 + 2 + 2 + 2 + 0.5 = 10.5s (+8s)
```

### Total Sync Time Estimation

```
Total Time = (C Ã— T_avg) + (R_count Ã— T_retry)

Where:
  C = Total contacts
  T_avg = Average time per successful contact (2.5s)
  R_count = Number of contacts requiring retry
  T_retry = Additional time for retry (4s)

Example: 50 contacts, 2 retries
  Total Time = (50 Ã— 2.5) + (2 Ã— 4)
```
